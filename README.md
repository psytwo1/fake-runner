# FakeRunner
FakeRunner uses idevicelocation to send location information written to iOS device triggered by writing to Firestore

# System Requirement
- idevicelocation
- Mac
- iOS device
  
# Getting started
1. Edit config,json

`
{
  "userId": "user is here",
  "udid": "device UDID here"
}
`

2. Build

`npm run build`

3. Run

`npm start`
